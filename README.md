# DailyEmbodiedWorks

A curated, daily-updated digest of recent research on Vision-Language-Action
(VLA) models and broader Embodied AI. Each daily log organizes papers into
thematic buckets:

- VLA Model Design (architectures, tokenization, efficiency)
- Instruction Following & Language Grounding
- Memory & Long-Horizon Reasoning
- World-Model-based VLA
- Latent Action Embedding / Action Representation
- Reinforcement Learning for VLA (Post-Training, RL Infrastructure)
- Benchmarks, Simulators & Evaluation
- Applications (Driving, UAV, Humanoid, etc.)

Highlighted works are flagged with **[Highlight]** when they are, in the
curator's view, particularly well-motivated, well-designed, or likely to have
broad impact on the field.
